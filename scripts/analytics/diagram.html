Why ADLS Gen2 raw data matters
Hive Partitioning
Blobs stored under eventType / year / month / day / hour / stationId. Only the hours in your query window are scanned — no full-table reads.
Replayable History
Raw events are immutable. Re-run any analysis against any past window without touching the live pipeline — ideal for backtesting new metrics.
Any Tool Can Read It
The same blobs power Databricks, Synapse Analytics, Azure Data Factory, or a local Python script. No proprietary format lock-in.
Cost-Effective Storage
Gzip-compressed JSON (~370 bytes / event). 70 k events ≈ 26 MB stored. Lifecycle policies tier cold partitions to archive automatically.
Fine-Grained Access Control
POSIX-style ACLs at every path level. Grant a team read access to eventType=arrival/ without exposing eventType=line-status/.
Big-Data ETL Ready
The Hive layout is natively understood by Spark, Presto, and Azure Synapse. Scale from a local script to a distributed cluster with zero schema changes.
